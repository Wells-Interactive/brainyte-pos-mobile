## 📱 Overview

**Brainyte POS Mobile App** is a modern, mobile-first Restaurant Point of Sale system built with **Flutter** and powered by **PHP**.

Designed for speed and simplicity, it enables restaurant staff to manage tables, place orders, and communicate instantly with the kitchen and bar using a clean and intuitive interface.

---

## ✨ Features

### 👨‍🍳 Waiter Module

* 🔐 Email & Password-based login
* 🍽️ Fixed restaurant tables (1–20)
* ➕ Create customer orders
* 🍔 Add unlimited food and drink items
* 📤 Send orders instantly
* 🔄 Live order status updates
* 📋 View active orders
* 💰 Close customer tables
* 📋 View active orders
* 🖨️ Thermal Printing
---

### 🍳 Kitchen Module

* 📡 Realtime incoming food orders
* ⏳ Pending Orders
* 👨‍🍳 Preparing Orders
* ✅ Ready for Pickup
* ⚡ Automatic updates to waiters

---

### 🍹 Bar Module

* 📡 Live drink orders
* ⏳ Pending
* 🍸 Preparing
* ✅ Ready
* 🔄 Instant synchronization
